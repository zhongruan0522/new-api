The legacy `GET /api/oauth/wechat/bind` description is outdated.

Current repository code does not expose that old GET bind endpoint at runtime, so integrations should not rely on it.

If WeChat login is enabled, follow the active `/api/oauth/wechat` login flow and the current frontend integration instead of the removed GET bind callback.
