現在のランタイム実装は `POST /api/oauth/email/bind` です。

この API は現在ログイン中のアカウントにメールアドレスを紐付けるためのもので、旧 GET クエリ形式は受け付けません。

リクエストボディ:

```json
{
  "email": "user@example.com",
  "code": "123456"
}
```

補足:

- 有効なログイン済みセッションが必要です。
- 検証コードは事前に `/api/verification` で同じメールアドレス向けに発行されている必要があります。
- 旧 `GET /api/oauth/email/bind?email=...&code=...` は使用しないでください。
