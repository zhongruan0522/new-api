旧 `GET /api/oauth/wechat/bind` の説明は現在の実装と一致していません。

現在のリポジトリコードでは、その旧 GET バインドエンドポイントはランタイム公開されていないため、この旧 API に依存しないでください。

WeChat ログインを利用する場合は、現行の `/api/oauth/wechat` フローと実際のフロントエンド実装を参照してください。
