旧版 `GET /api/oauth/wechat/bind` 文档描述已经与当前实现不一致。

当前仓库代码在运行时并没有暴露这条旧的 GET 绑定接口，因此集成方不应继续依赖这个旧回调地址。

如果启用了微信登录，请以当前仍存在的 `/api/oauth/wechat` 登录流程和实际前端接入方式为准，而不是继续调用已移除的旧 GET 绑定接口。
